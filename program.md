PROGRAM JANGKA PENDEK DAN JANGKA PANJANG
CALON WALI NAGARI AIE TAJUN LUBUK ALUNG
ARSINDO IRMAWAN, S.T
No. Urut 3


1. PEMERINTAHAN NAGARI BERBASIS DIGITAL

Program Jangka Pendek
1. Pembuatan sistem administrasi nagari berbasis digital.
2. Pengadaan website dan media informasi nagari.
3. Pelatihan dasar teknologi informasi bagi aparatur nagari.
4. Penyediaan sarana pendukung digitalisasi pelayanan.
5. Peningkatan pelayanan publik yang cepat, mudah, dan transparan.

Program Jangka Panjang
1. Pengembangan Smart Nagari berbasis teknologi terintegrasi.
2. Digitalisasi penuh pelayanan administrasi masyarakat.
3. Sistem data nagari terpadu dan transparan.
4. Penguatan tata kelola pemerintahan profesional dan akuntabel.
5. Pusat layanan informasi publik berbasis online.

2. KEMANDIRIAN EKONOMI NAGARI BERBASIS POTENSI LOKAL

Program Jangka Pendek
1. Penguatan kelompok tani dan usaha masyarakat.
2. Pelatihan UMKM dan kewirausahaan masyarakat.
3. Pengembangan produk unggulan nagari.
4. Pendampingan pemasaran hasil usaha masyarakat.
5. Optimalisasi peran BUMNag sebagai penggerak ekonomi.

Program Jangka Panjang
1. Terwujudnya nagari mandiri pangan dan ekonomi.
2. Pusat ekonomi kreatif dan UMKM berbasis lokal.
3. Pengembangan kawasan usaha produktif masyarakat.
4. Penguatan BUMNag menjadi lembaga ekonomi nagari yang maju.
5. Peningkatan kesejahteraan masyarakat melalui ekonomi berkelanjutan.

3. MASYARAKAT NAGARI YANG RELIGIUS DAN BERAKHLAK

Program Jangka Pendek
1. Dukungan kegiatan TPA/TPSA dan pendidikan Al-Qur’an.
2. Pembinaan generasi muda melalui kegiatan keagamaan.
3. Penguatan kegiatan masjid dan mushalla.
4. Program pembinaan akhlak dan kepedulian sosial.
5. Dukungan terhadap lembaga dan kegiatan keagamaan.

Program Jangka Panjang
1. Terwujudnya generasi muda religius dan berkarakter.
2. Penguatan nilai moral dan kehidupan sosial harmonis.
3. Nagari yang religius, aman, dan berakhlak mulia.
4. Pengembangan pusat kegiatan keagamaan masyarakat.
5. Pelestarian nilai-nilai adat basandi syarak, syarak basandi Kitabullah.

4. PELESTARIAN ADAT DAN BUDAYA NAGARI

Program Jangka Pendek
1. Pembinaan kegiatan adat dan budaya nagari.
2. Dukungan kegiatan seni dan tradisi lokal.
3. Pelibatan niniak mamak, alim ulama, cadiak pandai, dan organisasi kemasyarakatan.
4. Pelestarian aset budaya dan sejarah nagari.
5. Kegiatan budaya untuk generasi muda.

Program Jangka Panjang
1. Terwujudnya masyarakat yang mencintai adat dan budaya.
2. Penguatan identitas budaya Nagari Aie Tajun.
3. Pengembangan pusat seni dan budaya nagari.
4. Pelestarian tradisi dan kearifan lokal secara berkelanjutan.
5. Regenerasi pelaku seni dan budaya daerah.

5. GENERASI MUDA AKTIF, KREATIF, DAN BERPRESTASI

Program Jangka Pendek
1. Pembinaan organisasi kepemudaan dan olahraga.
2. Perbaikan sarana olahraga nagari.
3. Kegiatan seni, kreativitas, dan sosial kepemudaan.
4. Pelatihan keterampilan dan kewirausahaan pemuda.
5. Penguatan semangat gotong royong generasi muda.

Program Jangka Panjang
1. Terwujudnya generasi muda inovatif dan mandiri.
2. Pengembangan pusat kegiatan pemuda dan olahraga.
3. Penciptaan pemuda berprestasi di tingkat daerah dan nasional.
4. Penguatan ekonomi kreatif berbasis pemuda.
5. Pembentukan generasi penerus nagari yang unggul dan berdaya saing.

6. PEMBANGUNAN NAGARI PARTISIPATIF RANAH DAN RANTAU

Program Jangka Pendek
1. Penguatan komunikasi masyarakat nagari dan perantau.
2. Program gotong royong masyarakat.
3. Pelibatan perantau dalam pembangunan nagari.
4. Penguatan kegiatan sosial kemasyarakatan.
5. Peningkatan kepedulian sosial dan kebersamaan masyarakat.

Program Jangka Panjang
1. Terwujudnya pembangunan nagari berbasis partisipasi masyarakat.
2. Kolaborasi kuat antara ranah dan rantau.
3. Pengembangan jaringan ekonomi dan sosial perantau.
4. Budaya gotong royong yang berkelanjutan.
5. Nagari maju melalui sinergi pemerintah, masyarakat, dan perantau.
